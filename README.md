# Bridge leaderboards — 2025 World Championships

Four-facet card-play analysis for every player at the 2025 World Bridge Championships
(Herning), built from the WBF results microsite. Two pages:

- **[Declarer](index.html)** — every declarer with ≥40 boards, ranked.
- **[Defender](defenders.html)** — every partnership with ≥40 defended boards, ranked,
  with the IMP result split by which partner was on opening lead.

The four facets (declarer / defence mirror):
- **F1** tricks vs double-dummy
- **F2** tricks vs the field *in the same contract* on the same deal
- **F3** making / defeating the contract vs the field's rate for it
- **F4** the same-contract trick difference scored in **IMPs** (the ranking)

Each deal is scored from the actual result and compared to every other table that
played it. Expand the worked example at the top of either page to see every board for
one player/pair, reconciled to their totals. Static pages — no dependencies.
