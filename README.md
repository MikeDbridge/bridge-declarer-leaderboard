# Bridge leaderboards — World Championships

Four-facet card-play analysis for every player at the World Bridge Championships,
built from the WBF results microsites. Four pages (two years × two roles), cross-linked:

| | Declarer | Defender |
|---|---|---|
| **2025 (Herning)** | [index.html](index.html) | [defenders.html](defenders.html) |
| **2023 (Marrakech)** | [declarer-2023.html](declarer-2023.html) | [defenders-2023.html](defenders-2023.html) |

The four facets (declarer / defence mirror):
- **F1** tricks vs double-dummy
- **F2** tricks vs the field *in the same contract* on the same deal
- **F3** making / defeating the contract vs the field's rate for it
- **F4** the same-contract trick difference scored in **IMPs** (the ranking)

Declarers are ranked individually; defenders as partnerships, with the F4 result split
by which partner was on opening lead. Each deal is scored from the actual result and
compared to every other table that played it. Expand the worked example at the top of
any page to see every board for one player/pair, reconciled to their totals. Static
pages, no dependencies.
