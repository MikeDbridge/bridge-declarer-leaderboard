# Declarer leaderboard — 2025 World Bridge Championships

A four-facet analysis of declarer card-play for every player (≥40 boards) across the
four divisions of the 2025 World Bridge Championships (Herning), built from the WBF
results microsite.

- **F1** tricks vs double-dummy
- **F2** tricks vs the field *in the same contract* on the same deal
- **F3** making the contract vs the field's make-rate for it
- **F4** the same-contract trick difference scored in **IMPs** (the ranking)

Each deal is scored from the actual result and compared to every other table that
played it. Expand the worked example at the top of the page to see every board for
one player, reconciled to their totals.

Single static page — open `index.html`. Live at the repository's GitHub Pages URL.
